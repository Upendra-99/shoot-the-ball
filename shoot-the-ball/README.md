# Shoot The Ball 🌇
  It's a game developed in React.js with beautifull UI and responsiveness.

## Tech Stack Used 👩‍💻
- React
- Redux
- Styled-components

## Run Locally 💻


1. Install nodejs 

2. Clone the project 
   git clone https://github.com/Upendra-99/shoot-the-ball
   
3. run npm install
 
4. run npm start
